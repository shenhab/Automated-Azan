Environment Variables

To enable WhatsApp notifications via Twilio, create a .env file in your project directory with the following variables: 

# Twilio Configuration
TWILIO_ACCOUNT_SID=your_account_sid_here
TWILIO_AUTH_TOKEN=your_auth_token_here
TWILIO_CONTENT_SID=your_content_sid_here
TWILIO_WHATSAPP_NUMBER=whatsapp:+14155238886
RECIPIENT_NUMBER=whatsapp:+your_phone_number_here
These settings are configured outside the web interface for security reasons.
