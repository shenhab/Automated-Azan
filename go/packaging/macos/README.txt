INSTALL
───────
1. Drag AzanAgent  →  Applications  (in this window)

2. Open Terminal (double-click the Terminal shortcut in this window)
   then paste this command and press Enter:

   xattr -rd com.apple.quarantine /Applications/AzanAgent.app

3. Double-click AzanAgent in your Applications folder to launch.
   The icon will appear in your menu bar (top-right of screen).

That's it — the app installs itself as a background service and
starts automatically on every login.
